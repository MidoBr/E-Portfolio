In this section I will perform scan exercises against group 2 website using the tools available in Kali Linux. To answer the following questions as you can:

  • What Operating System does the web site utilize?
  • What web server software is it running?
  • What protection does it have (CDN, Proxy, Firewall?)

  • Does it have any open ports?
  • Does the site have any known vulnerabilities?
