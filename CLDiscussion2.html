Collaborative Learning Discussion 2 - Group 3 Summary Post

hanks to the peer Nils Linhoff and his team, we could deepen our understanding of the Internet Control Message Protocol (ICMP) and the User Diagram Protocol (UDP). The timeouts of Ping the server and the 64 hops of Traceroute derived from the security configuration of the target host, which enhanced the actual security by reducing the attack surface. Ping uses ICMP to communicate with the target server, and Traceroute uses ICMP and UDP to communicate with the same (Davis et al., 2006). As sophisticated cloud services like Amazon Web Services, internal monitoring tools are supported, enabling the least inbound and outbound traffic to enhance security. Disabling ICMP traffic is a reasonable approach to protect from ICMP flood attacks (Deepika, 2016). Another peer reviewer Mathew Van Beek provided sound advice to restrict using the nslookup because the Internet Systems Consortium (ISC) does not recommend it due to its frequent inconsistent behaviour documented in the Administrator Reference Manual (ISC, 2020). I agree with this recommendation, and even the latest Administrator Reference Manual (ARM) advises the same (ISC, 2021). As per the current standard, I would suggest removing "nslookup" from the Unit 3 "Practical and Team activity" page, mentioning "Perform a basic scan using standard tools such as ping, traceroute, dig and nslookup" (Computer Department, 2021).
